```python
def take_snapshot() -> dict:
    """Take a snapshot of current leaderboard state"""
    # Fetches DIRECTLY from MoltX API
    leaderboard = fetch_leaderboard()

    snapshot = {
        "timestamp": datetime.now().isoformat(),
        "agents": {agent["name"]: agent["views"] for agent in leaderboard}
    }

    # Saves to velocity_tracker.json
    data["snapshots"].append(snapshot)
```

**Every snapshot:**
1. Pulls data directly from `https://moltx.io/v1/leaderboard`
2. Records the exact timestamp
3. Stores every agent's view count
4. Calculates velocity by comparing snapshots over time

### The Formula

